A draggable slider component for selecting numeric values within a range. Built on Radix UI Slider for accessible keyboard navigation. Supports single value and range selection.

## Installation

<Tabs items={['CLI', 'Manual']}>
  <Tab value="CLI">
    ```bash
    npx shadcn@latest add @nteract/slider
    ```
    <RegistrySetup />
  </Tab>
  <Tab value="Manual">
    Install the dependencies:
    ```bash
    npm install radix-ui
    ```

    Copy the component from this repository's `registry/primitives/slider.tsx`.
  </Tab>
</Tabs>

## Usage

```tsx
import { Slider } from "@/registry/primitives/slider"

export function Example() {
  return <Slider defaultValue={[50]} max={100} step={1} />
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `number[]` | — | Controlled value |
| `defaultValue` | `number[]` | — | Default value |
| `onValueChange` | `(value: number[]) => void` | — | Callback when value changes |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `disabled` | `boolean` | `false` | Whether the slider is disabled |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Slider orientation |
| `className` | `string` | — | Additional CSS classes |
