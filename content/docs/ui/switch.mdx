A toggle switch component for boolean settings. Built on Radix UI Switch for accessible keyboard navigation and screen reader support. Supports default and small sizes.

## Installation

<Tabs items={['CLI', 'Manual']}>
  <Tab value="CLI">
    ```bash
    npx shadcn@latest add @nteract/switch
    ```
    <RegistrySetup />
  </Tab>
  <Tab value="Manual">
    Install the dependencies:
    ```bash
    npm install radix-ui
    ```

    Copy the component from this repository's `registry/primitives/switch.tsx`.
  </Tab>
</Tabs>

## Usage

```tsx
import { Switch } from "@/registry/primitives/switch"

export function Example() {
  return <Switch />
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `"sm" \| "default"` | `"default"` | Switch size |
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Default checked state |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when checked state changes |
| `disabled` | `boolean` | `false` | Whether the switch is disabled |
| `className` | `string` | — | Additional CSS classes |
