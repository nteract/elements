A multi-line text input component for forms, markdown cells, comments, and any content that requires multiple lines. Features automatic height adjustment and consistent styling.

## Installation

## Usage

```tsx
import { Textarea } from "@/registry/primitives/textarea"

export function Example() {
  return <Textarea placeholder="Enter your text..." />
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `placeholder` | `string` | — | Placeholder text |
| `rows` | `number` | — | Number of visible text lines |
| `disabled` | `boolean` | `false` | Disable the textarea |
| `className` | `string` | — | Additional CSS classes |
| `...props` | `TextareaHTMLAttributes` | — | All standard textarea attributes |

## Features

- **Auto-sizing**: Uses `field-sizing-content` for automatic height adjustment based on content
- **Consistent styling**: Matches Input component styling for visual consistency
- **Validation states**: Supports `aria-invalid` for form validation
- **Focus states**: Clear focus indicators with ring and border color changes
- **Dark mode**: Proper contrast and styling in dark mode

## Accessibility

- Supports all standard ARIA attributes
- Focus-visible styles for keyboard navigation
- Proper contrast in both light and dark modes
- Works with Label component for proper accessibility
